# 🔥 C Programming Practice Repository

This repository contains various **C programming problems and solutions** organized topic-wise.  
It is created to improve **problem solving skills, logic building, and understanding of core C concepts**.

---

## 📚 Topics Covered

| Topic | Description |
|------|-------------|
| Basic Problems | Beginner level C programs |
| 1D Array Problems | Programs based on single dimensional arrays |
| 2D Array Problems | Matrix and multidimensional array programs |
| Function Problems | Programs using user defined functions |
| Pattern Problems | Star and number pattern printing programs |
| Recursion Problems | Problems solved using recursion |
| String Problems | Programs related to string manipulation |
